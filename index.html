How "everyone can contribute" works: the app loads the shared dataset.json from the repo so you start from everyone's motions. When you click Contribute, your recorded data is copied to your clipboard + downloaded, and GitHub opens a new file for you under contributions/ — paste, commit, and it opens a Pull Request. Once a maintainer merges it, a GitHub Action rebuilds dataset.json automatically, so the base library grows for everyone. No tokens, no backend.
Honest note: there's no meaningful set of 700,000 distinct hand gestures, and from-scratch training needs many samples per category — start with a handful and let the community grow it.
